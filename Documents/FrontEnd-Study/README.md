공부한 내용을 Github, 블로그 등을 통해 정리하여 디스코드 채널에 공유한다. Javascript의 경우 구글링시 되도록 2년 이내의 자료를 참고하고 도서, 강의와 같이 다양한 자료도 참고하는 것이 좋다.

- 수요일: 팀원 서로의 결과물에 대해 질의응답

- 목요일: 자신의 결과물 보충

- 금요일: 쉬는 날
